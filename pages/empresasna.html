empresas nacionales

01

Microsoft Corporation

Microsoft es una empresa multinacional con sede en Estados Unidos que es conocida por su amplia gama de productos de software, incluyendo el sistema operativo Windows, la suite de productividad Office, la plataforma de desarrollo .NET, y servicios en la nube como Microsoft Azure.

02

Google LLC

Google es otra empresa estadounidense conocida por su software, incluyendo el motor de búsqueda Google Search, el sistema operativo móvil Android, el navegador web Chrome, la suite de productividad Google Workspace, y servicios en la nube como Google Cloud Platform.

03

Oracle Corporation

Oracle es una empresa de tecnología con sede en Estados Unidos que se especializa en software de base de datos y soluciones empresariales. Ofrecen una amplia gama de productos, incluyendo la base de datos Oracle, Oracle Cloud, y aplicaciones empresariales como Oracle ERP y Oracle CRM.

04

SAP SE

SAP es una empresa multinacional con sede en Alemania que es líder en software de planificación de recursos empresariales (ERP) y software de gestión empresarial. Ofrecen soluciones para diferentes industrias y procesos de negocio, incluyendo SAP S/4HANA, SAP SuccessFactors y SAP Ariba.

05

Salesforce.com, Inc.

Salesforce es una empresa estadounidense conocida por su software de gestión de relaciones con clientes (CRM). Ofrecen una plataforma en la nube para la gestión de ventas, marketing, servicio al cliente y análisis de datos, conocida como Salesforce Customer 360.
